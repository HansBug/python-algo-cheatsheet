## 项目概述

这是一个专为求职笔试和面试准备的 Python 算法模板库。所有代码都是为了让开发者快速记忆和应用而设计的，不是作为第三方库使用。

**核心特点：**
- 充分利用 Python 语言特性（列表推导、切片、内置函数等）
- 针对性能瓶颈优化，避免常见的速度陷阱
- 代码精简，方便理解和记忆
- 纯原生库实现，支持 Python 3.7-3.14
- 所有模板都经过实际题目验证

## 仓库结构

```
python-algo-cheatsheet/
├── algos/              # 算法模板存放区
│   ├── heap/          # 堆算法模板
│   │   ├── README.md          # 堆算法笔试攻略
│   │   ├── heapq_native.py    # 使用 heapq 的实现（推荐）
│   │   └── heap_manual.py     # 手动实现堆
│   └── [其他算法]/    # 未来会添加更多算法
├── tools/             # 通用工具库（非算法模板）
└── README.md          # 项目说明
```

### 目录说明

**algos/** - 算法模板核心目录
- 每个子目录代表一个算法类别（如 heap、dfs、dp 等）
- 每个算法目录包含：
  - `README.md` - 该算法的笔试攻略内容（必读）
  - 若干 `.py` 文件 - 可直接运行的算法代码示例
- **阅读顺序**：先看 README.md 了解概念和策略，再看代码实现

**tools/** - 通用工具库
- 用于支撑特定功能（如性能对比的 benchmark 等）
- 不是算法模板的一部分，读者可以忽略

## 算法模板编写规范

当添加新算法模板时，必须遵循以下规范：

### 1. 目录结构
```
algos/[算法名称]/
├── README.md           # 笔试攻略（必需）
├── [实现方式1].py      # 代码实现（至少一个）
└── [实现方式2].py      # 可选的其他实现方式
```

### 2. README.md 内容结构

每个算法的 README.md 应包含以下部分：

```markdown
# [算法名称]笔试攻略

## 什么是[算法]？
- 基本概念和定义
- 核心性质
- 支持的操作及时间复杂度

## [算法]适合解决什么问题？
- 列举典型应用场景
- 说明时间复杂度优势

## 笔试场景建议
### 推荐策略
- 优先使用标准库还是手动实现
- 各种方案的优缺点

## 方案一：使用标准库（如适用）
- 基本用法
- 常见技巧
- 代码示例

## 方案二：手动实现
- 适合速记的实现
- 使用示例
- 完整实现参考

## 常见题型速查
- 列举典型题型
- 提供代码模板

## 实战技巧总结
- 关键要点列表

## 参考资料
- 链接到本目录的代码文件
```

### 3. Python 代码文件规范

- **文件头注释**：说明模板用途、支持的 Python 版本、核心原理
- **代码风格**：简洁易记，充分利用 Python 特性
- **注释**：关键步骤添加注释，说明时间复杂度和原理
- **示例代码**：在 `if __name__ == '__main__':` 中提供完整的使用示例
- **兼容性**：确保代码在 Python 3.7-3.14 上都能运行

### 4. 命名规范

- 目录名：使用小写字母和下划线（如 `heap`、`binary_search`）
- Python 文件名：描述性命名（如 `heapq_native.py`、`heap_manual.py`）
- README.md：固定名称，必须大写

## 常用命令

```bash
python algos/heap/heap_manual.py
```

### 测试代码兼容性
```bash
# 使用不同 Python 版本测试
python3.7 algos/heap/heapq_native.py
python3.8 algos/heap/heapq_native.py
python3.12 algos/heap/heapq_native.py
```

## 开发注意事项

1. **纯原生库**：不要引入第三方依赖，确保 OJ 平台兼容性
2. **版本兼容**：避免使用 Python 3.8+ 专属特性（如海象运算符 `:=`）
3. **性能优化**：针对 Python 的性能特点优化，避免常见陷阱
4. **代码简洁**：优先考虑可读性和可记忆性，而非过度优化
5. **实战验证**：所有模板应在实际算法题中验证过
