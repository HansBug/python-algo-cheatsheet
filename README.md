> 专为求职刷题准备的 Python 算法模板库：简短、好记、快速

## 📖 这是什么？

这是一个专注于**求职笔试和面试**的 Python 算法模板集合。所有代码都是为了**让你背下来**而设计的，不是让你当第三方库调用的。

### 核心特点

- ✅ **发挥 Python 优势**：充分利用 Python 语言特性（列表推导、切片、内置函数等）
- ✅ **规避 Python 劣势**：针对性能瓶颈优化，避免常见的速度陷阱
- ✅ **简短易记**：代码精简，方便理解和记忆，考场上能快速写出
- ✅ **兼容性强**：纯原生库实现，支持 Python 3.7-3.14，保证 OJ 平台兼容
- ✅ **实战导向**：所有模板都经过实际题目验证

## 🎯 适合谁？

### ✅ 你应该来看这个库，如果你：

- 有一定 Python 基础，熟悉常用语法和数据结构
- 经常用 Python 写实用代码（脚本、数据处理等）
- 了解基本算法概念（排序、搜索、动态规划等）
- 准备用 Python 参加笔试/面试，但没怎么用 Python 刷过算法题
- 想快速掌握 Python 算法模板，提高刷题效率

### ❌ 你不应该来看这个库，如果你：

- 还不熟悉 Python 基础语法（变量、函数、类、列表、字典等）
- 不了解基本算法知识（时间复杂度、递归、图论等）
- 需要从零开始学习算法

**如果你属于上述情况，建议先去：**
- Python 基础：[廖雪峰 Python 教程](https://www.liaoxuefeng.com/wiki/1016959663602400) 或 [Python 官方教程](https://docs.python.org/zh-cn/3/tutorial/)
- 算法基础：[LeetCode 学习计划](https://leetcode.cn/studyplan/) 或《算法图解》

## 📚 内容结构

> 🚧 **施工中** - 内容正在持续更新

计划包含以下模板分类：

- 基础数据结构（栈、队列、堆、并查集等）
- 搜索算法（DFS、BFS、二分查找等）
- 动态规划（背包、LIS、LCS 等）
- 图论算法（最短路径、拓扑排序等）
- 字符串算法（KMP、字典树等）
- 数学与位运算技巧
- 常用技巧与优化

## 🚀 使用方式

1. 浏览对应分类的模板代码
2. 理解代码逻辑和适用场景
3. **背下来**（是的，就是背）
4. 在实际题目中应用和调整

## ⚠️ 注意事项

- 所有代码使用 Python 原生库，不依赖第三方包
- 代码兼容 Python 3.7-3.14，避免使用高版本专属特性
- 模板需要根据具体题目灵活调整，不是万能的
- 性能优化已考虑，但极端情况下可能仍需针对性优化
