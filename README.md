> 专为求职刷题准备的 Python 算法模板库：简短、好记、快速

## 📖 这是什么？

这是一个专注于**求职笔试和面试**的 Python 算法模板集合。所有代码都是为了**让你背下来**而设计的，不是让你当第三方库调用的。

### 核心特点

- ✅ **发挥 Python 优势**：充分利用 Python 语言特性（列表推导、切片、内置函数等）
- ✅ **规避 Python 劣势**：针对性能瓶颈优化，避免常见的速度陷阱
- ✅ **简短易记**：代码精简，方便理解和记忆，考场上能快速写出
- ✅ **兼容性强**：纯原生库实现，支持 Python 3.7-3.14，保证 OJ 平台兼容
- ✅ **实战导向**：所有模板都经过实际题目验证

## 🎯 适合谁？

### ✅ 你应该来看这个库，如果你：

- 有一定 Python 基础，熟悉常用语法和数据结构
- 经常用 Python 写实用代码（脚本、数据处理等）
- 了解基本算法概念（排序、搜索、动态规划等）
- 准备用 Python 参加笔试/面试，但没怎么用 Python 刷过算法题
- 想快速掌握 Python 算法模板，提高刷题效率

### ❌ 你不应该来看这个库，如果你：

- 还不熟悉 Python 基础语法（变量、函数、类、列表、字典等）
- 不了解基本算法知识（时间复杂度、递归、图论等）
- 需要从零开始学习算法

**如果你属于上述情况，建议先去：**

- Python 基础：[廖雪峰 Python 教程](https://www.liaoxuefeng.com/wiki/1016959663602400)
  或 [Python 官方教程](https://docs.python.org/zh-cn/3/tutorial/)
- 算法基础：[LeetCode 学习计划](https://leetcode.cn/studyplan/) 或《算法图解》

## 📚 已收录算法

> 按**从常见到不常见**排序；常见度只做粗粒度区分，同一层内再按**从简单到困难**排序。

| 算法                                                      | 简介                          | 使用建议                                      |
|---------------------------------------------------------|-----------------------------|-------------------------------------------|
| [堆（Heap）](algos/heap/README.md)                        | 完全二叉树结构，支持 O(log n) 插入和删除最值 | 优先使用 `heapq` 标准库，适合 Top K、优先级队列、动态维护最值等问题 |
| [并查集（Union-Find）](algos/union_find/README.md) | 处理不相交集合的合并与查询，支持 O(α(n)) ≈ O(1) 操作 | 极简版只需 10 行代码，适合连通性判断、环检测、最小生成树、等价类划分等问题 |
| [二分图匹配（匈牙利算法）](algos/bipartite_matching/README.md) | 在二分图中寻找最大匹配，时间复杂度 O(V × E) | 手动实现约 20 行代码，适合任务分配、配对问题、棋盘染色、最小点覆盖等问题 |
| [平衡树（Balanced Tree）](algos/balanced_tree/README.md) | 自平衡的二叉搜索树，保证 O(log n) 操作效率 | 笔试推荐 Treap（代码简洁，性能稳定），避免红黑树，适合动态有序集合、区间查询、排名查询等问题 |

## ⚠️ 注意事项

- 所有代码使用 Python 原生库，不依赖第三方包
- 代码兼容 Python 3.7-3.14，避免使用高版本专属特性
- 模板需要根据具体题目灵活调整，不是万能的
- 性能优化已考虑，但极端情况下可能仍需针对性优化
- 部分代码编写过程中会使用 Claude Code 提高效率，但所有算法核心代码都经过本人完全优化，确保适合面试场景

Python 但没用它刷过题"的人。
