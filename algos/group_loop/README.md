# 分组循环笔试攻略

## 什么是分组循环？

分组循环就是用两个指针把 **连续相同的一段** 一次性处理掉，而不是一个元素一个元素地硬分类讨论。

- 常见写法是 `i` 指向组头，`j` 一路扫到组尾
- 每轮处理区间 `[i, j)`
- 处理完令 `i = j`
- 这玩意在字符串压缩、连续段统计、相同值分块里特别顺手

### 支持的操作

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 扫描分组 | O(n) | 每个元素只进出一组 |
| 统计连续段 | O(n) | 例如最长连续重复字符 |
| 压缩输出 | O(n) | 例如 `aaabb -> a3b2` |

## 分组循环适合解决什么问题？

1. **连续相同字符 / 数字分段**
    - 例如：字符串压缩、RLE 编码
    - 时间复杂度：O(n)

2. **统计连续段长度**
    - 例如：最长重复字符段、连续块个数
    - 时间复杂度：O(n)

3. **排序后按值分组**
    - 例如：把相同值放在一起批量处理
    - 时间复杂度：排序 O(n log n) + 分组 O(n)

## 笔试场景建议

### 推荐策略

1. **只要题目在按“连续相同的一段”处理，优先写分组循环**
    - ✅ 代码比层层 `if` 清楚
    - ✅ 不容易漏最后一组

2. **固定模板记住 `while i < n`**
    - ✅ 每轮找 `j`
    - ✅ 处理中间这一组
    - ✅ 最后 `i = j`

3. **如果题目要求相同元素相邻，先排序再分组**
    - ✅ 这是很常见的组合套路

## 方案一：通用模板（推荐）

```python
i = 0
n = len(nums)

while i < n:
    j = i + 1
    while j < n and nums[j] == nums[i]:
        j += 1

    # 处理区间 [i, j)

    i = j
```

## 常见题型速查

### 1. 统计每一段

```python
i = 0
while i < len(nums):
    j = i + 1
    while j < len(nums) and nums[j] == nums[i]:
        j += 1
    ans.append((nums[i], j - i))
    i = j
```

### 2. 字符串压缩

```python
parts = []
i = 0

while i < len(s):
    j = i + 1
    while j < len(s) and s[j] == s[i]:
        j += 1
    parts.append(s[i] + str(j - i))
    i = j
```

### 3. 最长连续重复段

```python
ans = 0
i = 0

while i < len(s):
    j = i + 1
    while j < len(s) and s[j] == s[i]:
        j += 1
    ans = max(ans, j - i)
    i = j
```

## 实战技巧总结

✅ **记住核心骨架**：`i` 找组头，`j` 找组尾

✅ **每轮处理的是 `[i, j)` 整组**

✅ **处理完一定 `i = j`**

✅ **最后一组天然会被处理到**：别额外补特殊分支

✅ **题目要求相同元素连续时，先排序再分组**

✅ **连续段问题别写成一堆 if-else**
