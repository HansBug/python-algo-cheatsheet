# 差分笔试攻略

## 什么是差分？

差分可以看成前缀和的逆操作，特别适合处理 **多次区间加减**。

- `diff[i]` 表示当前位置相对前一位的变化量
- 区间 `[l, r]` 加 `val`：`diff[l] += val`，`diff[r + 1] -= val`
- 最后跑一遍前缀和就能还原数组
- 如果更新很多、查询最终结果一次，差分非常香

### 支持的操作

| 操作 | 时间复杂度 | 说明 |
|------|------------|------|
| 单次区间更新 | O(1) | 只改两个端点 |
| 批量更新 | O(m) | m 次更新 |
| 还原结果 | O(n) | 对差分做前缀和 |

## 差分适合解决什么问题？

1. **多次区间加法**
    - 例如：批量更新数组区间
    - 时间复杂度：O(n + m)

2. **乘客上下车 / 航班预订**
    - 例如：某段区间统一加人数或座位
    - 时间复杂度：O(n + m)

3. **覆盖统计**
    - 例如：每个位置被覆盖多少次
    - 时间复杂度：O(n + m)

4. **配合离散化处理大坐标**
    - 例如：坐标很大但更新次数不多
    - 时间复杂度：O(m log m)

## 笔试场景建议

### 推荐策略

1. **只要是“很多次区间加减，最后统一看结果”，优先想差分**
    - ✅ 这是差分最典型的使用场景
    - ✅ 每次更新从 O(length) 变成 O(1)

2. **先统一边界定义**
    - ✅ 是闭区间 `[l, r]` 还是半开区间 `[l, r)` 要先想清楚
    - ❌ 边界没统一最容易错

3. **最后别忘了还原**
    - ✅ 差分数组本身不是答案

## 方案一：通用模板（推荐）

```python
diff = [0] * (n + 1)

for left, right, val in updates:
    diff[left] += val
    if right + 1 < n:
        diff[right + 1] -= val

ans = [0] * n
cur = 0
for i in range(n):
    cur += diff[i]
    ans[i] = cur
```

## 常见题型速查

### 1. 批量区间加法

```python
diff = [0] * (n + 1)
for left, right, val in updates:
    diff[left] += val
    if right + 1 < n:
        diff[right + 1] -= val
```

### 2. 还原最终数组

```python
cur = 0
for i in range(n):
    cur += diff[i]
    ans[i] = cur
```

### 3. 拼车 / 预订统计

```python
for num, start, end in trips:
    diff[start] += num
    diff[end] -= num  # 这里用了左闭右开的站点区间
```

## 实战技巧总结

✅ **多次区间更新优先想差分**

✅ **单次更新只动两个端点**

✅ **最后必须做一次前缀和还原**

✅ **边界定义先统一**：闭区间和半开区间别混

✅ **大坐标题常和离散化一起用**

✅ **只改最终结果时很强，频繁在线查询时不适合**
